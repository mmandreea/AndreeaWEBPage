## 🎯 Overview

Pagina Personala is a modern, responsive personal portfolio website designed to present professional work, technical skills, and accomplishments in an engaging and accessible format.

This website is hosted on Vercel: https://andreeamata.vercel.app/

## ✨ Features

- Responsive design optimized for desktop, tablet, and mobile devices
- Professional portfolio showcase
- About section with detailed background
- Skills and competencies listing
- Project gallery with descriptions
- Contact information and links
- Fast loading and smooth navigation

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- Responsive Web Design

## 💻 Local Installation & Setup

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Sublime, etc.) - for development

### Installation Steps

1. Clone the repository:
```bash
git clone <https://github.com/mmandreea/AndreeaWEBPage.git>
cd PaginaPersonala
```

2. Open the project:
```bash
# Option 1: Direct file opening
open index.html

# Option 2: Using a local server 
python -m http.server 8000
```

3. Access in browser:
- Direct: Open `index.html` in your browser
- Local server: Navigate to `http://localhost:8000`
